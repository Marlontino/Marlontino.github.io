Image Manipulation Project

A Python wrapper script that integrates with MATLAB, C, Haskell, and Prolog to manipulate images

Project Overview

This project demonstrates multi-language programming and image processing techniques by creating a Python wrapper that orchestrates image manipulation operations across four different programming languages: MATLAB, C, Haskell, and Prolog.

Technologies Used

Python MATLAB C Haskell Prolog

Key Features

  • Image Resizing: MATLAB code resizes photos and converts them to 1D arrays
  • Grayscale Conversion: C program converts images to grayscale
  • Value Flipping: Haskell program flips the pixel values
  • Image Rotation: Prolog performs 180-degree rotation
  • Python Orchestration: Wrapper script coordinates all operations

How It Works

  1. The Python wrapper script initiates the process
  2. MATLAB takes the original image and resizes it, outputting to "input.txt"
  3. C program reads "input.txt" and converts to grayscale, creating "c_output.txt"
  4. Haskell flips the values from "input.txt"
  5. Prolog performs 180-degree rotation on the original image
  6. Final MATLAB script reads all output files and displays the results

Project Structure

  • wrapper.py - Main Python orchestrator
  • matlabcode1.m - Image resizing and array conversion
  • matlabcode2.m - Final display and visualization
  • CProg.c - Grayscale conversion
  • HProg.hs - Value flipping
  • PProg.pl - Image rotation

Getting Started

To run this project, you'll need Python, MATLAB, gcc, Haskell, and Prolog installed on your system. Simply run:

python3 wrapper.py
