Stock Analyzer

A comprehensive stock market analysis and trading system with momentum analysis, volatility detection, and ML-based signals

Project Overview

A comprehensive stock market analysis and trading system that provides momentum analysis, volatility detection, sector rotation insights, and machine learning-based trading signals. This project demonstrates advanced data analysis, financial modeling, and machine learning skills applied to real-world market data.

Technologies Used

Python Machine Learning Pandas SQLite yfinance scikit-learn Financial Analysis

Key Features

  • Momentum Analysis: Identify stocks with strong price momentum and risk-adjusted returns (Sharpe ratio)
  • Volatility Detection: Find stocks with unusual volatility patterns (breakouts/consolidation)
  • Pairs Trading: Sector-specific pairs trading opportunities with correlation analysis
  • Sector Rotation: Comprehensive sector performance analysis across all 11 S&P 500 sectors
  • ML Signals: Machine learning-based buy/sell/hold signals using voting classifiers
  • CLI Interface: Easy-to-use command-line interface for all operations

How It Works

  1. Data Collection: Fetches stock data from Yahoo Finance via yfinance
  2. Data Storage: Stores data in SQLite database for efficient querying
  3. Analysis Pipeline: Runs momentum, volatility, pairs, and sector analyses
  4. ML Predictions: Generates buy/sell/hold signals using ensemble classifiers
  5. Reporting: Outputs actionable insights for investment decisions

Installation & Usage

# Clone the repository
$ git clone https://github.com/Marlontino/Stock-Analyzer.git
$ cd stock-analyzer

# Create virtual environment (Python 3.10+ required)
$ python3 -m venv venv
$ source venv/bin/activate

# Install dependencies
$ pip install -r requirements.txt

# Run all analyses
$ ./stock-analyzer analyze

CLI Commands

# Check database status
$ ./stock-analyzer status

# Update all S&P 500 stock data
$ ./stock-analyzer update

# Run specific analysis
$ ./stock-analyzer analyze momentum
$ ./stock-analyzer analyze volatility
$ ./stock-analyzer analyze pairs
$ ./stock-analyzer analyze sectors
$ ./stock-analyzer analyze ml
