Chess Game

A fully-featured chess game built with Python and Pygame, supporting Player vs Player and Player vs CPU with AI.

Project Overview

This project is a complete chess game built in Python using Pygame. It features both Player vs Player and Player vs CPU modes, a beautiful menu system, sound effects, multiple board themes, and a greedy AI opponent. The game implements all standard chess rules, including castling, en passant, pawn promotion, and check/checkmate detection.

Technologies Used

Python Pygame Game Development AI User Interface

Key Features

  • Player vs Player and Player vs CPU modes
  • Greedy AI opponent
  • Full chess rules: move validation, check, checkmate, stalemate
  • Pawn promotion, en passant, castling
  • Sound effects for moves and captures
  • Multiple board themes (press 'T' to change)
  • Visual move indicators and last move highlighting
  • Beautiful menu system with hover effects
  • Drag and drop piece movement
  • Win condition detection and restart options

How to Play

# Clone the repository
$ git clone https://github.com/Marlontino/Chess.git
$ cd Chess

# (Optional) Create a virtual environment
$ python -m venv venv
$ source venv/bin/activate # On Windows: venv\Scripts\activate

# Install dependencies
$ pip install -r requirements.txt

# Run the game
$ python src/main.py

All images and screenshots are included in the images/ folder inside the Chess project.

Game Controls

  • Mouse: Click and drag pieces to move
  • T: Change board theme
  • R: Restart current game
  • M: Return to main menu
  • Q: Quit application

AI Opponent

The CPU opponent uses a greedy algorithm that evaluates all possible moves, considers piece values, and chooses the move that maximizes material advantage. The AI plays as black in PvC mode.
