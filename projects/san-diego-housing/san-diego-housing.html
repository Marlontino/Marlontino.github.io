Comprehensive data analysis of the San Diego Airbnb market with predictive modeling and interactive visualizations

Project Overview

A comprehensive data analysis project examining the San Diego Airbnb market using advanced analytics, predictive modeling, and interactive visualizations. This project provides deep insights into pricing patterns, neighborhood analysis, and market trends to help understand the rental market dynamics.

Features

  • Data Cleaning: Automatic outlier removal and data quality checks
  • Price Analysis: Statistical analysis with clean vs. original data comparisons
  • Neighborhood Insights: Top neighborhoods by listings, pricing, and value scores
  • Market Composition: Room type distribution and pricing analysis
  • Predictive Modeling: Random Forest regression with feature importance analysis
  • Interactive Maps: Geographic heatmaps and clickable neighborhood markers
